Come funzionano i wizard nella nuova TUI?

Ogni wizard (OCR, Comprimi, Verifica) richiede più passaggi: inserire il file, scegliere opzioni, confermare l'output. Come vuoi che vengano mostrati questi passaggi?

Passo 1 di 4
┌─ Esegui OCR ──────────────────────────────┐
│ Percorso del PDF di input │
│ ▌ /Users/mario/documenti/scan.pdf │
│ │
│ Invio continua Esc annulla │
└───────────────────────────────────────────┘
→ Invio: si apre il passo successivo (lingua OCR)
→ ogni passo sostituisce il precedente

A — Schermate a passi

Ogni passo è una nuova schermata full-screen. Invio avanza, Esc torna indietro. Il wizard "cammina" step-by-step. Approccio più simile all'attuale.

┌─ Esegui OCR ──────────────────────────────┐
│ 1. File ▶ 2. Lingua 3. Formato 4. Output │
├───────────────────────────────────────────┤
│ │
│ Lingua OCR │
│ > Italiano │
│ English │
│ Italiano + English │
│ │
└───────────────────────────────────────────┘
→ barra in cima mostra sempre a che punto sei
→ si può tornare a un passo precedente cliccando

B — Stepper con barra progresso

Un'unica schermata del wizard con indicatore di passo in cima. Il contenuto cambia al passo corrente ma il contesto (titolo, barra) rimane visibile.

┌─ Home ────────────────────────────────────┐
│ > Strumento OCR │
│ Comprimi PDF │
│┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄│
│ File ▌ scan.pdf │
│ Lingua ● Italiano ○ Eng ○ Ita+Eng │
│ Output ▌ scan.1.pdf │
│ [ Avvia OCR ] │
└───────────────────────────────────────────┘
→ il form appare nella metà inferiore della home
→ nessun cambio di schermata, tutto in un'unica vista

C — Form inline nella home

Selezionando un'azione, il form appare direttamente nella schermata principale (sotto il menu). Nessuna navigazione tra schermate — tutto in vista. Più compatto, meno "cinematico".
